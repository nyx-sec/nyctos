# Verb form of "leverage": followed by a determiner / possessive so the
# noun forms ("leverage point", "leverage ratio") survive.
regex:\bleverage[s]?\s+(the|this|that|these|those|our|their|your|its|my|a|an|all|both|users|customers|developers|teams|community)\b

# Marketing-speak "robust": followed by an abstract noun the word is
# usually padding for. Legitimate concrete uses ("robust testing",
# "robust parser") are not flagged.
regex:\brobust\s+(system|systems|platform|platforms|solution|solutions|framework|frameworks|approach|approaches|architecture|infrastructure|methodology|design|capability|capabilities)\b

# Unqualified "powerful": followed by an abstract / marketing-speak noun,
# or appearing at end of clause (period / comma / em-dash). Concrete
# noun uses ("powerful API", "powerful tool") survive.
regex:\bpowerful\s+(features?|capabilit(y|ies)|abstractions?|toolkits?|solutions?|combinations?|engines?)\b
regex:\bpowerful[[:space:]]*[.,;!?]
